# Jednostavni Notepad

A simple text editor application inspired by basic notepad functionality, developed as a programming project to practice core software development concepts.

# Features
Create, open, and edit text files
Save text content to local files
Basic text manipulation
Simple and intuitive user interface
# How to Run
-Clone the repository:
git clone https://github.com/borna-deroncico/Jednostavni-notepad
-Navigate to the project folder:
cd Jednostavni-notepad
-Run the application:
python main.py
